## Sovelluksen asennus ja käyttäminen

Riippuvuuksien asennus:

`poetry install`

Alusta sovellus:

`poetry run invoke build`

Käynnistä sovellus:

`poetry run invoke start`

Testien ajaminen:

`poetry run invoke test`

Testikattavuus:

`poetry run invoke coverage-report`
